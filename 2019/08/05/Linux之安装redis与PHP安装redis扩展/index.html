Linux-安装redis与PHP安装redis扩展
Linux
Linux
发布日期:   2019-08-05
文章字数:   302
阅读时长:   1 分
阅读次数:  

1、安装 gcc

sudo apt install gcc

2、下载并解压

$ wget http://download.redis.io/releases/redis-5.0.5.tar.gz
$ tar xzf redis-5.0.5.tar.gz
$ cd redis-5.0.5
$ make

## 确定结束的时候没有报错：error 如果报错了别慌张复制错误代码谷歌百度，慢慢解决确认没问题后
$ make install

安装后的目录和文件看图说话：

创建配置目录复制配置文件

$ sudo mkdir /etc/redis

4、创建持久保持数据文件和创建启动服务

创建持久储存数据文件

$ sudo mkdir /var/lib/redis

先修改 redis.conf 配置文件

新建 Redis 服务进程系统启动文件

sudo gedit /etc/systemd/system/redis.service

[Unit]
Description=Redis In-Memory Data Store
After=network.target

[Service]
User=redis
Group=redis
ExecStart=/usr/local/bin/redis-server /etc/redis/redis.conf
ExecStop=/usr/local/bin/redis-cli shutdown
Restart=always

[Install]
WantedBy=multi-user.target

PHP 安装 Redis 扩展

$ wget https://pecl.php.net/get/redis-5.0.2.tgz
$ tar -zxf redis-5.0.2.tgz
$ cd redis-5.0.2
$ phpize

$ ./configure --with-php-config=/usr/local/php/bin/php-config

记住了 -config 后面跟的是你自己 php-config 目录

$ make && make install

编译之后会在当前redis扩展这个目录下面生成一个 modeles 目录 里面会有个 redis.so
