Manjaro Linux

最强大最牛逼最好的 Linux 桌面发行版本，不接受任何的反驳！！！我是 KDE ，其他版本感觉上面都没有 KDE舒服！反正我感觉是最好用的版本！！！

Linux 桌面系统排行榜：https://distrowatch.com/

为什么会选择 Manjaro 这个发行版本呢？

1、滚动升级

2、自动硬件检测安装

3、先进的包管理、最多软件支持的 Linux 发行版本

4、超级高度用户自定义、桌面、主题、界面、文件、操作、显示等等，全部可用户自定义

5、软件版本超级牛逼的同步快，基本与各个软件的版本持平

6、目前生态最好的一款 Linux 桌面版本之一（在我心里是第一）

7、本地化相比 18.10 之前已经做的非常好了，我又倒腾了目前 19.0.2 的 KDE 版本

8、超级稳定

中科大软件源

1.选择源
sudo pacman-mirrors -i -c China -m rank 
运行一段时间后，会出现选择框，这里选择
http://mirrors.ustc.edu.cn/manjaro

2.刷新缓存
sudo pacman -Syy

3.修改配置文件
sudo vim /etc/pacman.conf
添加到末尾
[archlinuxcn]
SigLevel = Optional TrustedOnly
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

4.导入 GPG Key
sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring

Cisco AnyConnect

### 如果安装了 Cisco 没有问题无需操作,如果有问题请按照下面重新操作
# 1、注释用户验证
sudo vim vpn_install.sh 
# Make sure we are root
# if [ `id | sed -e 's/(.*//'` != "uid=0" ]; then
#   echo "Sorry, you need super user privileges to run this script."
#   exit 1
# fi

# 2、创建 rc.d
sudo mkdir /etc/rc.d

# 3、安装,如果还有权限 root 允许
./vpn_install.sh

# 4、启动服务
sudo /etc/rc.d/vpnagentd start

https://bbs.archlinux.org/viewtopic.php?id=190444
