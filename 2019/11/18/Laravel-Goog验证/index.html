Laravel-Goog验证
Laravel-扩展包
Laravel
发布日期:   2019-11-18
文章字数:   449
阅读时长:   1 分
阅读次数:  

摘要

对于一些项目敏感操作来说验证用户都是属于必备的一个环节，当然验证的方式很多，确实很多！Google 验证，在我看来是验证当中的一种不错的选择，毕竟一次操作终身可行，而且 Google 自身就帮我们处理好了验证码有效期。（每30秒刷新一次），签名算法安全可靠！Laravel 做为 PHP 目前使用率最高的框架，当然也有他的 Google 验证扩展包。

安装

pragmarx/google2fa-laravel

# 扩展 Imagick 开启
composer require pragmarx/google2fa-laravel

bacon/bacon-qr-code

# 扩展 iconv 开启
# dasprid/enum
composer require bacon/bacon-qr-code

安装 Imagick 扩展

apt-get install imagemagick
convert -version 使用这个命令检测是否安装成功，如果成功的话会有imagemagick版本号。
apt-get install libmagickwand-dev

配置

生成 Google2FA 配置文件，文件存在位置 config/google2fa.php，命令如下：

php artisan vendor:publish --provider="PragmaRX\Google2FALaravel\ServiceProvider"

使用

生成 Secret Key

use PragmaRX\Google2FA\Google2FA;

$google2fa = new Google2FA();
$secretKey = $google2fa->generateSecretKey(64);

# 这里的 Key 就是我们需要验证的 Key

获取二维码地址

$company = ''; // 公司名称
$holder = ''; // 用户标示，可以是邮箱或者名称等
$qrcodeUrl = $google2fa->getQRCodeUrl($company, $holder, $secretKey);

生成二维码图片

// 图片路径
$filePath = sha1($secretKey) .'.png';

// 生成图片
$renderer = new ImageRenderer(
   new RendererStyle(200),
   new ImagickImageBackEnd()
);
$writer = new Writer($renderer);
$writer->writeFile($qrcodeUrl, $filePath);

最后，通过使用 Google Authenticator 应用程序来扫描二维码，扫描完毕之后，就会生成一组 6 位的数字密码。

身份验证

$google2fa = new Google2FA();
$verify = $google2fa->verifyKey($secretKey, $request->input('code'));
if ($verify) {
    return true;  // 验证成功
}
return false;
