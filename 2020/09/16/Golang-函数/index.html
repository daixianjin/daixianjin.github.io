基本了解

首先什么是函数呢？为了完成某一功能的程序指令（语句）的集合称为函数，在 Go 语言当中函数分为：自定义函数和系统函数。在 PHP 当中函数属于一等公民，毫无例外在 Go 语言当中函数也是属于一等公民，所以说函数是相当重要的角色。Go 语言函数对比 PHP 的函数的特殊之处在于他可以 return 带有零个或多个参数；这些参数将作为返回值，而 PHP 做不到多返回值（返回一个数组，却还是单一返回值），别说 PHP 大多数语言都无法做到多返回值的效果。

Go 里面有三种类型的函数：

  • 普通的带有名字的函数
  • 匿名函数或者闭包函数
  • 方法（Methods）

注意：除了 main () 主函数、init () 初始函数外，其它所有类型的函数都可以有参数与返回值。函数参数、返回值以及它们的类型被统称为函数签名。Go 中没有引用传值，包括 slice/map/chan 在内所有类型，没有传引用的说法，传指针也是将指针的值拷贝一份，可以确认的是 Go 语言中所有的传参都是值传递（传值），都是一个副本，一个拷贝。因为拷贝的内容有时候是非引用类型（int、string、struct 等这些），这样就在函数中就无法修改原内容数据；有的是引用类型（指针、map、slice、chan 等这些），这样就可以修改原内容数据。是否可以修改原内容数据，和传值、传引用没有必然的关系。在 C++ 中，传引用肯定是可以修改原内容数据的，在 Go 语言里，虽然只有传值，但是我们也可以修改原内容数据，因为参数是引用类型。

语法

fun name (参数列表) (返回值列表) {
        执行的语句...
        return 然后值列表
}

# 细节，如果没有返回值那么不用写括号，如果只有一个返回值也不用写括号，多个返回值必须有括号括起来，另外注意：函数中的变量是局部变量，函数外不生效

快速入门

package main

import "fmt"

func main() {
    result := sum(10, 20)
    fmt.Printf("结果为：%d", result)
}

func sum(a int, b int) int {
    return a + b
}

结果：

结果为：30

函数调用和细节

package.函数名称(参数列表)

注意：这里 package 也就是包了，当然同一个包当中不需要加上 package 就跟上面的快速入门一样，不同包之间需要加上 package，比起 PHP->说实话我更喜欢 . 因为只需要按一个键位！

  • 在调用一个函数时，会给该函数分配一个新的空间，编译器会通过自身的处理，这个新的空间和其他的栈的空间区分开来。
  • 在每个函数对应的栈中，数据空间是隔离的，不会混淆。
  • 当一个函数执行完毕后，程序会销毁这个函数对应的栈空间。

函数使用注意事项

函数的参数列表可以是多个，返回值列表页可以是多个，并且支持对函数返回值命名

package main

import "fmt"

func main() {
    a, b := sum(1, 2, 3)
    fmt.Println("他们的值", a, b)
}

func sum(a int, b int, c int) (res int, err string) {
    return a + b + c, "你好世界"
}

// 结果：他们的值 6 你好世界

函数的命名遵循标识符命名规范，首字母不能是数字，首字母大写该函数可以被本包文件和其它包文件使用，类似 public ，首字母小写，只能被本包文件使用，其他包文件不能使用，类似 privat

package main

import "fmt"

func main() {
    test()
    Test()
}

func test() {
    fmt.Println("只能本包使用")
}

func Test() {
    fmt.Println("可以其它包使用")
}

不支持函数重载（PHP 也不支持函数的重载）,但可以使用别的方式实现（空接口）

package main

import "fmt"

func main() {
    test()
}

func test(a int) int {
    return a
}

func test(a int, b int) int {
    return a
}

// 错误，不支持传统的函数重载，但是可以使用别的方式去实现

函数也是一种数据类型，可以赋值给一个变量，则该变量就是一个函数类型的变量，通过该变量可以对函数进行调用

package main

import "fmt"

func main() {
    b := test
    res := b(1)
    fmt.Printf("res=%v", res)
}

func test(a int) int {
    return a
}

// 结果：res = 1

函数既然是一种数据类型，因此在 Go 中，函数可以作为形参，并且调用

package main

import "fmt"

func main() {
    a := sumFunc(sum, 1, 2)
    fmt.Printf("a = %v\n", a)
}

func sum(a int, b int) int {
    return a + b
}

func sumFunc(a func(a int, b int) int, a1 int, a2 int) int {
    return a(a1, a2)
}

// 结果：a = 3

为了简化数据类型定义，Go 支持自定义数据类型，这个在 C 语言当中就有，别的一般语言基本没有这么一个特性，也可以把它理解成为别名，这样就容易理解这个自定义数据类型

语法：

type 自定义数据类型名 数据类型
type myInt int // 那么现在的 myInt 就是 int 的一个别名，但是 Go 从语法中认为它们还是不同的二个类型

type myFunc func(a int,b int) int // 一个自定义的函数类型

Go 支持可变参数的传递，它的原理就是 args 是 slice 切片，通过 args[index] 可以访问到各个值

package main

import "fmt"

func main() {
    fmt.Printf("sum() = %v", sum(10, 10, 10, 10, 10))
}

func sum(a int, args ...int) int {
    sum := a

    for i := 0; i < len(args); i++ {
        sum += args[i]
    }

    return sum
}

// 结果：sum() = 50

函数递归

一个函数在函数体内又调用了本身，那么就称之为函数递归。Go 语言支持递归，但在使用递归时，需要设置退出条件，否则递归将陷入无限循环。

快速入门

package main

import "fmt"

func main() {
    test(4)
}

func test(n int) {
    if n > 2 {
        n--
        test(n)
    }
    fmt.Printf("当前 n 的值为 %d \n", n)
}

结果

当前 n 的值为 2 
当前 n 的值为 2 
当前 n 的值为 3

详解

细节

  • 递归必须有退出条件，否则无限循环调用
  • 执行一个函数时，就会创建一个新的受保护的独立空间（新函数栈）
  • 函数的局部变量都是独立的，不会互相影响
  • 当一个函数执行完毕，或者遇到 return 就会返回，尊崇谁调用，就将结果返回给谁，同时当函数执行完毕或者返回时，该函数本身也会自动被系统销毁

裴波那契数

package main

import "fmt"

func main() {
    result := f(4)
    fmt.Printf("求出的裴波那契数为：%v", result)
}

// 1 1 2 3 5 8 13...
func f(n int) int {
    if n == 1 || n == 2 {
        return 1
    } else {
        return f(n-1) + f(n-2)
    }
}

结果：

求出的裴波那契数为：3

defer

在函数中，需要基础创建资源（比如：连接数据库，文件句柄，锁等）为了在函数执行完毕后，及时的释放资源，Go 的设计者提供了 defer （延时机制），当程序执行到 defer 时，暂时不执行会将 defer 后面的语句压入到独立的栈中。当函数执行完毕后，再从 defer 栈，按照先入后出的方式执行。

快速入门

import "fmt"

func main() {
    sum(10, 20)
    fmt.Println("上面都执行完成了")
}

func sum(a int, b int) int {
    defer fmt.Printf("a 等于 %d \n", a)
    defer fmt.Printf("b 等于 %d \n", b)

    result := a + b
    fmt.Printf("a + b 的和是 %d \n", result)
    return result
}

结果：

a + b 的和是 30 
b 等于 20 
a 等于 10 
上面都执行完成了

详解

当程序执行到 defer 时，暂时不会将 defer 后面的语句执行出来，而是把他压入到 defer 栈当中。栈的特性相信大家都知道：先进后出和队列的特性完全相反。那么就会接着向下执行，执行到第三个 fmt 打印出 a + b 的和是 30接下 return 表示整个函数整个执行完毕了。此时，在回到 defer 栈当中把刚压入栈的取出来，先进后出的执行那么分别就是b 等于 20和a 等于 10。切记一点，defer 的语句会被压栈，执行时也是跟栈的特性一致。

注意：在 defer 语句放入栈时，也会将相关的值拷贝同时入栈，看以下例子。

细节

package main

import "fmt"

func main() {
    sum(10, 20)
    fmt.Println("上面都执行完成了")
}

func sum(a int, b int) int {
  //  defer 语句前改变了他们，那么他们会是改变的，因为在入栈前已经改变了
  a++
    b++
    defer fmt.Printf("a 等于 %d \n", a)
    defer fmt.Printf("b 等于 %d \n", b)
    // defer 语句后改变了他们，上面二个语句虽然在最后执行，但是在开始时就已经被压入栈了，随同一起压入的还拷贝的值，所有就算在他们之后改变了值也不会影响
  a++
    b++
    result := a + b
    fmt.Printf("a + b 的和是 %d \n", result)
    return result
}

结果：

# defer 语句前改变了他们 结果
a + b 的和是 32 
b 等于 21 
a 等于 11 
上面都执行完成了

# defer 语句后改变了他们 结果
a + b 的和是 32 
b 等于 20 
a 等于 10 
上面都执行完成了

defer 的实际应用场景

// 关闭文件流
defer file.Close()

// 解锁一个加锁的资源 
defer mu.Unlock()

// 关闭数据库链接
defer disconnectFromDB()

详解：Go 编程中通常的做法是创建资源后，如（打开了文件，连接了数据，锁资源）可以立即执行 defer 关闭资源。在 defer 后，可继续使用创建的资源。就不需要我们时刻去关注启用了资源，在什么适合的时机关闭资源而烦心，让程序更加的简洁。

init

每一个源文件都可以包含一个 init 函数，该函数会在 main 函数前执行，被 Go 运行框架调用，也就是说 init 会在 main 函数前被调用，通常它用来做一个初始化的操作。

快速入门

package main

import "fmt"

func init() {
    fmt.Printf("我是 init 函数，我现在被调用了 \n")
}

func main() {
    fmt.Printf("我是 main 函数，我现在被调用了 \n")
}

结果：

我是 init 函数，我现在被调用了 
我是 main 函数，我现在被调用了

细节

  1. 如果一个源文件同时包含全局变量定义、init 函数、main 函数，则执行的流程是：全局变量定义 > init 函数 > main 函数，它们的权重就是如此。

  2. init 函数不能有参数值，与返回值。

  3. 如果一个包引入了另外一个包，它们都有全局变量定义和 init 函数，执行流程会是怎么样？

匿名函数

Go 支持匿名函数，匿名函数就是没有名字的函数，如果我们某个函数只是希望使用一次，可以考虑使用匿名函数，匿名函数也可以实现多次调用。

语法

fun ( 参数列表) (返回值列表) {
        执行的语句...
        return 然后值列表
}(参数列表)

快速入门

package main

import "fmt"

var (
    // 定义全局的匿名函数 乘
    Multiply = func(a int, b int) int {
        return a * b
    }
)

func main() {
    // 在定义匿名函数时直接调用，这中种方式的匿名函数只能调用一次
    res1 := func(a int, b int) int {
        return a + b
    }(10, 20)
    fmt.Printf("res1 = %v\n", res1)

    // 在定义匿名函数时复制给一个变量，那么就可以反复去调用
    a := func(a int, b int) int {
        return a + b
    }
    res2 := a(1, 1)
    fmt.Printf("res2 = %v\n", res2)
    res3 := a(2, 3)
    fmt.Printf("res3 = %v\n", res3)

    // 使用全局的匿名函数
    res4 := Multiply(10, 10)
    fmt.Printf("res4 = %v\n", res4)
}

结果：

res1 = 30
res2 = 2
res3 = 5
res4 = 100
