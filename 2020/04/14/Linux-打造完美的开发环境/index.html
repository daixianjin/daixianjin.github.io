摘要

萝卜青菜各有所爱，Linux 桌面可以说常年是一个大乱斗的环境，谁都不服谁，也没有人能站出来统一天下，也不可能统一，毕竟开源自由是 Linux 的宗旨！这也算 Linux 的魅力之一吧！！！不同于 Windows 和 Mac 的软件生态环境，Linux 确实弱了那么一点，Linux 的软件就很少吗？不！他并不少！在我日常的开发和生活应用下，软件基本可以和另外二个一样够用，当然如果玩游戏的话那么 Windows 是不二的选择。

Linux 桌面系统排行榜：https://distrowatch.com/ ，前 10 我用过的还挺多，最终让我停留的还是 Manjaro ,它非常的吸引人。软件多，稳定、滚动更新升级、自定义强等等。我使用的版本 19.0.2 KDE 版本，除了 KDE Manjaro 提供了 2 个版本和众多社区版本！当然强力推荐 KDE 版本，当然这个也是看个人。

切换源

自动选择源，Manjaro 官方源

# 自动选择源
sudo pacman-mirrors -i -c China -m rank 

# 更新源
sudo pacman -Sy

手动配置软件源，非官方源 Arch Linux 国内源，中科大源

# 修改配置文件，添加配置到末尾
sudo vim /etc/pacman.conf

[archlinuxcn]
SigLevel = Optional TrustedOnly
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

# 导入 GPG Key
sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring

# 强制更新包数据，同时更新软件
sudo pacman -Syyu

初始化

初始化系统，其实就是安装驱动和语言包了，都是傻瓜式的一键安装，但是前提条件是必须系统为最新的状态，如有未更新需要更新后再操作。

找到系统设置->硬件设定->语言包。注：就算不安装驱动和语言包，也不会影响正常使用。

美化桌面

后续，图表、字体、光标都可以安装上面的操作进行，字体根据自己的需求可以调整，个人感觉默认的还是非常不错！

应用程序风格
