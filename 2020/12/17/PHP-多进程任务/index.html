PHP 多进程定时任务

摘要

PHP 实现多进程的方式很多，PHP 的扩展 pcntl 与 posix 双剑客。Workerman 也是的多进程也是基于这二个双剑客去实现滴，当然更为强大的 Swoole 就不需要它们了。最近项目有个需求，多进程定时去执行某一个任务，要求是精确到秒，定时任务写的还是比较多，首先想到的是 Crontab 虽然只能精确到分，但是利用 shell 脚本也可以实现秒级别的定时执行，但是想了下还需要多任务就算了吧，最终还是选定了 Supervisor 来实现多进程，关于 Laravel 如何实现定时任务，点我跳转。

多进程

Supervisor

关于 Supervisor 是什么也不过多介绍了，点我跳转，有详细的说明，对于 Supervisor 不是很熟悉也看下这篇文章熟悉熟悉，也可以点我跳转，也有介绍 Supervisor 的使用与一些需要注意的地方。

配置

[program:task]
process_name=%(program_name)s_%(process_num)02d
command=/usr/bin/php /app/artisan batch:store --id=%(process_num)02d --max=1
autostart=true
autorestart=true
user=root
numprocs=8 # 进程数量
redirect_stderr=true
stdout_logfile=/etc/supervisor/store.log

就这么简单，一个多进程的任务就算创建完成了，是不是非常简单。创建一个多进程的任务就是这么简单，但是如何管理好进程就是一件非常考人的问题了。我这里选择使用的 Redis 锁来管理进程。另外有个特别的注意点，那就是 Supervisor 的一个参数：startsecs 在启动后多少秒没有异常退出就表示正常的启动，默认为 1 秒。简单点来说，启动之后如果 1 秒都没到就退出执行了那么就表示异常，此时 Supervisor 就是报错，启动的 Supervisor 主进程就会挂逼。

在使用到了进程想一个问题，PHP 代码执行不到 1 秒就完成了退出了那么此时就是错误，还有进程它们是并行去执行 PHP 脚本，如果处理的数据并发有要求，那么肯定不可能同一时间就放所有进程去跑 PHP 脚本。Laravel 的队列也是使用到了锁，当这个进程去跑特定的一个队列时，第二个进程也进来了，此时这个进程获取不到锁，那么就进不去核心业务逻辑的代码，就马上会退出，其速度估计毫秒或者微妙之间就完成了，但是 Supervisor 的 startsecs 默认是 1 秒，1 秒都没有它就认为异常了，那么主进程就挂逼了，所以有个 sleep 的机制。这也是为啥说 sleep 非常重要了。

这也是我最近写多进程并发跑脚本发现的一个问题，踩了不少坑，如果有人也遇到这个问题了，可以看看 startsecs 这个参数的含义。多进程并发执行脚本，不依赖 PHP 的多进程扩展，依赖 Supervisor 来完成操作。

Redis 锁

<?php

namespace App\Services;

use Redis;

class LockService
{
    /**
     * Redis
     * */
    protected $redis;

    /**
     * Token
     * */
    protected $token;

    /**
     * prefix
     */
    protected $prefix = 'redis-lock-';

    /**
     * value
     * */
    protected const VALUE = 1;

    /**
     * 服务初始化,单机锁
     * @param Redis|null $redis
     */
    public function __construct(Redis $redis = null)
    {
        $this->redis = $redis ?: $this->createRedis();
    }

    /**
     * 创建 Redis 实例
     *
     * @return Redis
     */
    protected function createRedis(): Redis
    {
        $redis = new Redis();

        $redis->connect(config('database.redis.default.host'), config('database.redis.default.port'));
        $redis->auth(config('database.redis.default.password'));

        return $redis;
    }

    /**
     * 加锁
     *
     * @param string $key
     * @param int $ttl
     * @param int $retry
     * @return bool
     */
    public  function lock(string $key,int $ttl = 1,int $retry = 3) : bool
    {
        while (true) {

            if ($this->redis->set($this->prefix . $key,self::VALUE, ['NX', 'EX' => $ttl]))
            {
                return true;
            } elseif ($this->redis->ttl($this->prefix . $key) === -1) {
                // 以防出现死锁
                $this->redis->expire($this->prefix . $key,$ttl);
            }

            if ($retry-- <= 0) {
                break;
            }

            usleep(10000);
        }

        return false;
    }

    /**
     * 解锁
     *
     * @param string $key
     *
     * @return bool
     */
    public function unlock(string $key) : bool
    {
        if($this->redis->get($this->prefix . $key)) {
            $this->redis->multi();
            $this->redis->del($this->prefix . $key);
            $this->redis->exec();
            return true;
        } else {
            return false;
        }
    }
}

开始任务

任务模板自定义，因为使用了 Laravel 那么就使用 Laravel 的 command 来创建任务的模板。

<?php

namespace App\Console\Commands;

use Illuminate\Console\Command;
use App\Services\MultiProgressService;

class BatchSearchTask extends Command
{
    /**
     * The name and signature of the console command.
     *
     * @var string
     */
    protected $signature = 'batch:search
        {--id=      : 当前编号}
        {--max=     : 最大线程}
        ';

    /**
     * The console command description.
     *
     * @var string
     */
    protected $description = 'batch mode data search';

    /**
     * Create a new command instance.
     *
     * @return void
     */
    public function __construct()
    {
        parent::__construct();
    }

    /**
     * Execute the console command.
     *
     * @return mixed
     */
    public function handle()
    {
        // 初始化进程
        $service = new MultiProgressService(
            $this->option('id') ?? 00,
            $this->option('max') ?? 8
        );

        // 开始任务
        $service->start('search');
    }
}

处理任务

<?php

namespace App\Services;

use Carbon\Carbon;
use Webpatser\Uuid\Uuid;
use Illuminate\Support\Facades\DB;
use Illuminate\Support\Facades\Log;
use App\Facades\UserBatchSearchNumber;
use App\Facades\UserBatchSearchRecording;
use App\Services\V1\Front\SingleModeService;

class MultiProgressService
{
    /**
     * 任务编号
     *
     * @var int
     */
    protected $id;

    /**
     * 最大进程
     *
     * @var int
     */
    protected $max;

    /**
     * 单次服务
     * */
    protected $single;

    /**
     * 锁服务
     * */
    protected $lock;

    /**
     * 初始化服务
     *
     * @param int $id
     * @param int $max
     * */
    public function __construct(int $id,int $max)
    {
        $this->id = $id;
        $this->max = $max;
        $this->lock = new LockService();
        $this->single = new SingleModeService();
    }

    /**
     * 开始任务
     * @param string $drive
     * @param int $ttl
     * @return bool
     */
    public function start(string $drive = 'store',$ttl = 600) : bool
    {
        // 超过规定的进程数不做任何处理就返回
        if ($this->id >= $this->max) {
              usleep(1000 * 1000); // sleep 1 秒，因为 Supervisor 的 startsecs 默认就是一秒
            return true;
        }

          // 记录当前进程与驱动，作为 Key
        $name = sprintf('task:%03d:%s', $this->id, $drive);

        try {
            // 加锁成功进入逻辑执行
            if ($this->lock->lock($name,$ttl)) {
                              $this->search();
                $this->lock->unlock($name); // 执行完之后解锁
            }
        }catch (\Exception $e) {
            Log::channel('task')->debug($drive,[
                'line' => $e->getLine(),
                'message' => $e->getMessage(),
                'code' => $e->getCode(),
                'file' => $e->getFile(),
                'progress_id' => $this->id,
                'max_progress_number' => $this->max,
                'lock_name' => $name,
                'is_unlock' => $this->lock->unlock($name)
            ]);
        }

        // 悲观的情况下,因为 Supervisor 执行延时最低 1 秒需要验证进程是否还在运行，如果不是在运行那么会 Error，锁定 1 秒钟。
        usleep(1000 * 1000);

        return true;
    }
}
